# ai-rag-chatbot
AI chatbot using Endee and dataset
📌 Overview
This project is a Retrieval Augmented Generation (RAG) chatbot that uses Endee as a vector database to retrieve relevant information and generate accurate responses.
⚙️ System Design

User Query → Endee Vector Search → Retrieve Data → AI Response
🚀 Features
Semantic search using Endee
AI-generated answers
Context-based responses
Career guidance use-case
🧠 Use of Endee
Endee is used to:
Convert dataset into vectors
Perform semantic search
Retrieve relevant context
🛠️ Setup

git clone https://github.com/praveennp158-cell/ai-rag-chatbot
cd ai-rag-chatbot
pip install -r requirements.txt
python app.py
